# Vyra — Frontend

Aplicación Flutter multiplataforma (iOS, Android, Web, Linux, macOS, Windows).

## 🚀 Cómo empezar

1. Asegúrate de tener Flutter instalado:

   ```bash
   flutter doctor
   ```

2. Instala las dependencias:

   ```bash
   flutter pub get
   ```

3. Ejecuta la app:

   ```bash
   flutter run
   ```

## 📁 Estructura

```
lib/
├── core/           ← Temas, providers globales
├── features/       ← Módulos por feature (auth, home, posts, profile, search)
├── services/       ← Servicios de negocio
└── main.dart
```
